
Проект: Навигатор РЖДК — неоновая версия.

Файлы для загрузки в репозиторий:
- index.html
- rzhdk_images.json  (список изображений, можно редактировать)
- rzhdk_data.json    (данные зданий — загружается после экспорта из админки)
- /images/           (папка с изображениями, имя формата: {buildingId}_{floor}.jpg и schedule_...)

Инструкции:
1) Скопируй index.html в корень репозитория.
2) Добавь rzhdk_images.json (если хочешь) и заполните /images/ соответствующими файлами.
3) Открой сайт через GitHub Pages. Админка доступна по ?admin=secret (скрыта).
4) Редактируй здания, экспортируй rzhdk_data.json и залей его в репо, чтобы изменения увидели все.
